### Why colon

- `siwg`, `art-template`, `doT` ...

```html
<h1>{{ title }}</h1>
<ul>
{% for author in authors %}
    <li{% if loop.first %} class="first"{% endif %}>{{ author }}</li>
{% endfor %}
</ul>
```

- `colon`

```html
<h1>{{ title }}</h1>
<ul>
    <li :each="(author, index) in authors" :class="[index == 0 ? 'first' : '']">{{ author }}</li>
</ul>
```

`colon` has a more concise template syntax.
